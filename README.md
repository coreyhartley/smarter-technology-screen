# smarter-technology-screen
## requirements
Python >= 3.11

## install
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## run
```bash
python sort.py --length <length> --width <width> --height <height> --mass <mass>
```

## test
```bash
pytest test_sort.py
```
